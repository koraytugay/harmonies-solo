# Harmonies Solo

A beautiful web-based token drawing simulator for the board game Harmonies.

## Running at

http://koraytugay.github.io/harmonies-solo

## Features

- 🎲 Random token drawing with animated fade-in
- 🌊 Smooth falling animations when discarding tokens
- 📊 Animated progress bar showing remaining tokens
- 🎨 Clean, warm brown color palette
- ⌨️ Keyboard controls (Press Enter to draw)
- 🎯 Triangular token arrangement in three circular groups

## How to Use

1. Open the page
2. Press **Enter** to draw 9 random tokens
3. Tokens are displayed in 3 groups of 3
4. Press **Enter** again to discard and draw new tokens
5. Progress bar shows remaining tokens by color
